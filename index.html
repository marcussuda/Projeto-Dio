Falando sobre Tags

início

O que são as Tags

As tags são estruturas que contêm instruções e delimitam o início e o fim de uma seção.

As TAGS não aparecem na tela do computador, elas apenas servem como orientação.

Algumas TAGS tem a a TAG de abertura com o nome entre o caracteres <> e a TAG de fechamento com o nome entre o caracteres .

EX: < head >< /head >.

< strong >< /strong >.

< i >< /i >.

Outras TAGS, não tem TAGS de fechamento entre o caracteres <>.

EX: < img />

< imput />

Atributos das Tags

início

Os atridutos são as propriedades que as TAG's podem ter.

EX: TAG "id" que pode ser feito a identificação do texto, "style" determinar o estilo do texto.

Todos elementos HTML podem ter atributos Os atributos fornecem informações adicionais sobre os elementos Os atributos são sempre especificados na tag inicial Os atributos geralmente vêm em pares nome/valor como: nome="valor"

Listas Ordenadas e Não Ordenadas

início

Listas Ordanedas são as lista que começamos com a TAG "ol" e todos os itens que colocarmos na lista eles quando acompanhados da TAG "li" terão númeração sequencial.

Enquanto as Listas Não Ordenadas que são seguidas pela TAG "ul", todos os itens que colocarmos na lista quando os mesmos estiverem acompanhados da TAG "li", os itens terão apenas um ponto na frente.
